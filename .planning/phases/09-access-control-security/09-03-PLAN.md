# Stripe CSP Requirements
Stripe requires the following CSP directives:
- scriptSrc: js.stripe.com (for Stripe.js)
- frameSrc: js.stripe.com, checkout.stripe.com (for Stripe Checkout iframe)
- connectSrc: api.stripe.com (for API calls, already covered by default-src)
- formSrc: checkout.stripe.com (for payment form submission)
</context>
