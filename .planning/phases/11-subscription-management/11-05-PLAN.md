  <action>
Create Vue component at `app/components/subscription/PauseDialog.vue` that:

1. **Component props**:
   ```typescript
   const props = defineProps<{
     open: boolean
     currentPeriodEnd?: string // For showing warning about paid period
   }>()
   ```

2. **Emits definition**:
   ```typescript
   const emit = defineEmits<{
     (e: 'confirm', data: {indefinite: boolean, resumeDate?: string}): void
     (e: 'cancel'): void
   }>()
   ```

3. **Local state**:
   ```typescript
   const indefinite = ref(true) // Default to indefinite
   const resumeDate = ref('') // ISO date string

   // Computed for validation
   const canConfirm = computed(() => {
     return indefinite.value || (resumeDate.value && new Date(resumeDate.value) > new Date())
   })
   ```

4. **Confirm handler**:
   ```typescript
   function handleConfirm() {
     if (!canConfirm.value) return

     if (indefinite.value) {
       emit('confirm', { indefinite: true })
     } else {
       emit('confirm', { indefinite: false, resumeDate: resumeDate.value })
     }

     // Reset state
     indefinite.value = true
     resumeDate.value = ''
   }
   ```

5. **Cancel handler**:
   ```typescript
   function handleCancel() {
     emit('cancel')
     // Reset state
     indefinite.value = true
     resumeDate.value = ''
   }
   ```

6. **Template structure**:
   ```vue
   <template>
     <UModal v-model="props.open">
       <UCard>
         <template #header>
           <h3 class="text-lg font-semibold">Pausar Suscripción</h3>
         </template>

         <div class="space-y-4">
           <!-- Info alert -->
           <UAlert color="blue" icon="i-heroicons-information-circle" variant="soft">
             Al pausar la suscripción, se congela el facturación pero sigues teniendo acceso a QR.
           </UAlert>

           <!-- Indefinite toggle -->
           <div class="flex items-center justify-between">
             <div>
               <p class="font-medium">Pausar indefinidamente</p>
               <p class="text-sm text-gray-600 dark:text-gray-400">
                 La suscripción permanecerá pausada hasta que la reanudes manualmente
               </p>
             </div>
             <UCheckbox v-model="indefinite" />
           </div>

           <!-- Date picker (when not indefinite) -->
           <div v-if="!indefinite" class="space-y-2">
             <label class="block text-sm font-medium">Reanudar el:</label>
             <UInput
               v-model="resumeDate"
               type="date"
               :min="minDate()"
               placeholder="Selecciona una fecha"
             />
           </div>
         </div>

         <template #footer>
           <div class="flex justify-end gap-2">
             <UButton variant="ghost" @click="handleCancel">
               Cancelar
             </UButton>
             <UButton
               color="blue"
               :disabled="!canConfirm"
               @click="handleConfirm"
             >
               Pausar Suscripción
             </UButton>
           </div>
         </template>
       </UCard>
     </UModal>
   </template>
   ```

7. **Min date utility**:
   ```typescript
   function minDate() {
     const tomorrow = new Date()
     tomorrow.setDate(tomorrow.getDate() + 1)
     return tomorrow.toISOString().split('T')[0]
   }
   ```

**IMPORTANT CONTEXT DECISIONS:**
- Duration options: Both fixed date AND indefinite (CONTEXT.md: "Duration Options: Both fixed date AND indefinite")
- QR access during pause: Keep access (CONTEXT.md: "QR Access During Pause: Keep access")
- Default to indefinite (simpler UX)

**FOLLOW NUXT UI PATTERNS:**
Use UModal with v-model.bind for open/close, UCheckbox for toggle, UInput with type="date" for date selection.
