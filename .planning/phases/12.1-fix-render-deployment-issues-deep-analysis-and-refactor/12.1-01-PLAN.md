<task type="auto">
  <name>Task 2: Update render.yaml with optimized configuration</name>
  <files>render.yaml</files>
  <action>
    Update `render.yaml` with the following changes:
    
    1. **Update buildCommand** to use memory-optimized build:
       ```yaml
       buildCommand: npm ci --prefer-offline --no-audit && NODE_OPTIONS=--max-old-space-size=4096 npm run build
       ```
       This uses `--prefer-offline` for faster cached builds and `--no-audit` to skip audit checks. The NODE_OPTIONS sets 4GB memory limit for the Vite/Nuxt build process.
    
    2. **Add missing Mailgun environment variables** under envVars:
       ```yaml
       # Mailgun
       - key: API_KEY_MAILGUN
         sync: false
       - key: MG_DOMAIN
         sync: false
       ```
       These are required by `nuxt.config.ts` runtimeConfig.mailgun and currently missing from the Render blueprint.
    
    3. **Add runtime NODE_OPTIONS** (optional, for large operations during runtime):
       ```yaml
       - key: NODE_OPTIONS
         value: "--max-old-space-size=1536"
       ```
       This provides 1.5GB memory limit for runtime operations (half of the 2GB available on standard plan).
    
    Keep all existing configuration unchanged. Only add/modify the sections above.
  </action>
  <verify>
    <automated>grep -q "NODE_OPTIONS=--max-old-space-size=4096" render.yaml && grep -q "API_KEY_MAILGUN" render.yaml && grep -q "MG_DOMAIN" render.yaml && echo "OK" || echo "FAIL"</automated>
  </verify>
  <done>render.yaml contains memory-optimized build command and all required environment variables including API_KEY_MAILGUN and MG_DOMAIN</done>
