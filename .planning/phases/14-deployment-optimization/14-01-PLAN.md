# GSD Plan: Remove Phantom Dependencies

## Objective
Remove unused dependencies that bloat node_modules by ~84MB

## Problem
Several large packages are installed but not used:
- `three` (32MB) - 3D library, not used
- `lucide-vue-next` (29MB) - Using @iconify instead
- `better-sqlite3` (12MB) - Using MongoDB
- `playwright-core` (9.8MB) - Not used for testing

## Verification Commands
```bash
grep -r "from 'three'" app/ server/ --include="*.ts" --include="*.vue" || echo "three: NOT USED"
grep -r "from 'lucide-vue-next'" app/ server/ --include="*.ts" --include="*.vue" || echo "lucide: NOT USED"
grep -r "from 'better-sqlite3'" app/ server/ --include="*.ts" --include="*.vue" || echo "sqlite: NOT USED"
grep -r "from 'playwright'" app/ server/ --include="*.ts" --include="*.vue" || echo "playwright: NOT USED"
```

## Implementation

### Task 1: Verify packages are unused
Run grep commands above to confirm no imports exist.

### Task 2: Remove phantom dependencies
```bash
bun remove three lucide-vue-next better-sqlite3 playwright-core
```

### Task 3: Verify app still works
```bash
bun install
bun typecheck
bun dev  # Quick smoke test
```

## Success Criteria
- [ ] `three` removed from node_modules
- [ ] `lucide-vue-next` removed (keeping @iconify-json/lucide)
- [ ] `better-sqlite3` removed
- [ ] `playwright-core` removed
- [ ] node_modules reduced by ~80MB
- [ ] App starts without errors
- [ ] Typecheck passes

## Expected Savings
- **node_modules**: ~84MB reduction
- **Risk**: Low (packages verified unused)
