## Implementation

### Task 1: Clean install and measure
```bash
rm -rf node_modules .nuxt .output pnpm-lock.yaml
bun install
du -sh node_modules
```

### Task 2: Run build with memory limit
```bash
NODE_OPTIONS='--max-old-space-size=4096' bun run build
echo "Exit code: $?"
```

### Task 3: Measure output sizes
```bash
du -sh .output/server
du -sh .output/public
du -sh .output
```

### Task 4: List largest chunks
```bash
find .output/server -name "*.mjs" -exec du -h {} \; | sort -rh | head -20
```

### Task 5: Test application locally
