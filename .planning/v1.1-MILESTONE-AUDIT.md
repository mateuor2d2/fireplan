### 1. 🔴 DUPLICATE WEBHOOK ENDPOINTS (CRITICAL)

**Issue:** Two endpoints handle the same Stripe events:
- `server/api/webhooks/stripe.post.ts` (Phase 08) — Full implementation
- `server/api/payments/webhook.post.ts` (Phase 12) — Partial implementation

**Risk:** Double-processing, race conditions, inconsistent database state

**Affected:** All WEB-* requirements

**Resolution:** Consolidate to single endpoint

**Resolution:** Add CSP headers to nuxt.config.ts as documented in Phase 09-03
