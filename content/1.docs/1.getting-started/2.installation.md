## Requisitos Previos

Antes de comenzar, asegúrate de tener instalados los siguientes componentes:

- [Node.js](https://nodejs.org/) versión 18.x o superior
- [Bun](https://bun.sh/) (recomendado como gestor de paquetes)
- [MongoDB](https://www.mongodb.com/) versión 6.x o superior
- Cuenta en [AWS S3](https://aws.amazon.com/s3/) para almacenamiento de archivos
- Cuenta en [Stripe](https://stripe.com/) para procesamiento de pagos

## Configuración del Entorno de Desarrollo

### 1. Clonar el Repositorio

```bash [Terminal]
git clone https://github.com/tu-organizacion/prevenius.git
cd prevenius
```

### 2. Instalar Dependencias

Usamos Bun como gestor de paquetes por su velocidad y eficiencia:

```bash [Terminal]
bun install
```

### 3. Configurar Variables de Entorno

Crea un archivo `.env` en la raíz del proyecto con las siguientes variables:

```env
# MongoDB Connection
ME_CONFIG_MONGODB_URL=mongodb://localhost:27017/prevenius

# JWT Secret
JWT_SECRET=tu_secreto_jwt_aqui

# AWS S3 Configuration
AWS_ACCESS_KEY_ID=tu_access_key
AWS_SECRET_ACCESS_KEY=tu_secret_key
AWS_REGION=eu-west-1
AWS_BUCKET_NAME=prevenius-files

# Stripe Configuration
STRIPE_PUBLISHABLE_KEY=pk_test_tu_clave_publicable
STRIPE_SECRET_KEY=sk_test_tu_clave_secreta
STRIPE_WEBHOOK_SECRET=whsec_tu_secreto_webhook

# Mailgun Configuration
API_KEY_MAILGUN=tu_api_key_mailgun
MG_DOMAIN=prevenius.com

# OAuth Configuration
GOOGLE_CLIENT_ID=tu_client_id
GOOGLE_CLIENT_SECRET=tu_client_secret
GITHUB_CLIENT_ID=tu_github_client_id
GITHUB_CLIENT_SECRET=tu_github_client_secret
```

### 4. Iniciar el Servidor de Desarrollo

```bash [Terminal]
bun dev
```

El servidor estará disponible en `http://localhost:3000`.

## Scripts Disponibles

- `bun dev` - Inicia el servidor de desarrollo
- `bun build` - Compila la aplicación para producción
- `bun start` - Inicia el servidor en modo producción
- `bun lint` - Ejecuta el linter para verificar el estilo del código
- `bun typecheck` - Verifica los tipos de TypeScript

## Configuración de Base de Datos

### MongoDB Local

Para desarrollo local, puedes usar MongoDB Community Edition:

1. [Instala MongoDB](https://docs.mongodb.com/manual/installation/)
2. Inicia el servicio de MongoDB
3. La aplicación se conectará automáticamente a `mongodb://localhost:27017/prevenius`

### MongoDB Atlas (Recomendado para producción)

1. Crea una cuenta en [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
2. Crea un cluster gratuito
3. Configura las variables de entorno con tu cadena de conexión

## Configuración de Servicios Externos

### AWS S3

1. Crea una cuenta en AWS si no la tienes
2. Crea un bucket S3 para almacenar archivos
3. Crea un usuario IAM con permisos para el bucket
4. Configura las variables de entorno con tus credenciales

### Stripe

1. Regístrate en [Stripe](https://dashboard.stripe.com/register)
2. Obtén tus claves de API desde el dashboard
3. Configura los webhooks para eventos de pago
4. Actualiza las variables de entorno

## Resolución de Problemas

### Problemas Comunes

**Error de conexión a MongoDB:**
- Verifica que MongoDB esté en ejecución
- Confirma que la URL de conexión sea correcta
- Asegúrate de que no haya firewalls bloqueando el puerto

**Errores de dependencias:**
- Elimina `node_modules` y `bun.lock`
- Ejecuta `bun install` nuevamente

**Problemas con variables de entorno:**
- Verifica que el archivo `.env` exista en la raíz
- Asegúrate de que todas las variables requeridas estén definidas
