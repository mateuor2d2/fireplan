Las plantillas son una característica poderosa de Prevenius que te permite estandarizar y acelerar la creación de planes de seguridad. Esta guía te mostrará cómo aprovechar al máximo esta funcionalidad.

## Tipos de Plantillas

### Plantillas Globales

Las plantillas globales son proporcionadas por Prevenius y están disponibles para todos los usuarios:

- **Plantilla Estándar**: Formato básico con todos los elementos esenciales
- **Plantilla Detallada**: Versión completa con información extendida
- **Plantilla Compacta**: Formato resumido para impresión rápida
- **Plantilla Profesional**: Diseño empresarial con branding personalizado

### Plantillas Personalizadas

Puedes crear y guardar tus propias plantillas basadas en:

- Requisitos específicos de clientes
- Normativas locales o regionales
- Estilos corporativos
- Procedimientos internos

## Utilizar una Plantilla

### Seleccionar una Plantilla

1. Al generar un documento PDF, accede a la sección de impresión
2. Selecciona una plantilla de la lista disponible
3. Vista previa del contenido renderizado
4. Realiza ajustes si es necesario

### Personalizar Temporalmente

Puedes modificar aspectos de la plantilla para una generación específica:

- Cambiar colores y estilos
- Añadir logotipos o información de contacto
- Modificar encabezados y pies de página
- Ajustar el orden de las secciones

## Crear una Plantilla Personalizada

### Acceder al Gestor de Plantillas

1. Ve a la sección "Configuración" en tu perfil
2. Selecciona "Plantillas de Impresión"
3. Haz clic en "Nueva Plantilla"

### Estructura de una Plantilla

Las plantillas utilizan sintaxis de Handlebars para variables dinámicas:

```handlebars
# {{nom_obra}}

## Datos Generales
- **Dirección**: {{dir_obra}}, {{poblacion_obra}} ({{cp_obra}})
- **Duración**: {{duracion_meses}} meses
- **Presupuesto Total**: €{{presupuesto_total_obra}}

## Contratista
- **Empresa**: {{contratista.nom_contratista}}
- **CIF**: {{contratista.cif_contratista}}
```

### Variables Disponibles

#### Datos de la Obra
- `nom_obra` - Nombre de la obra
- `dir_obra` - Dirección de la obra
- `poblacion_obra` - Población
- `cp_obra` - Código postal
- `duracion_meses` - Duración estimada
- `presupuesto_total_obra` - Presupuesto total

#### Contratista
- `contratista.nom_contratista` - Nombre del contratista
- `contratista.cif_contratista` - CIF/NIF
- `contratista.dir_contratista` - Dirección
- `contratista.pob_contratista` - Población
- `contratista.cp_contratista` - Código postal
- `contratista.telf_contratista` - Teléfono
- `contratista.email_contratista` - Email

#### Promotor
- `nom_promotor` - Nombre del promotor
- `cif_promotor` - CIF/NIF
- `dir_promotor` - Dirección
- `localidad_promotor` - Localidad
- `cp_promotor` - Código postal
- `telf_promotor` - Teléfono
- `email_promotor` - Email

## Ayudas (Helpers) de Handlebars

Prevenius incluye ayudas personalizadas para facilitar la creación de plantillas:

### Formato de Fechas
```handlebars
{{formatDate createdAt "DD/MM/YYYY"}}
```

### Condicionales
```handlebars
{{#if contratista.telf_contratista}}
Teléfono: {{contratista.telf_contratista}}
{{/if}}
```

### Listas
```handlebars
{{#each partidas}}
- {{this.nombre}}: {{this.descripcion}}
{{/each}}
```

## Mejores Prácticas

### Organización de Plantillas

- **Nombra claramente**: Usa nombres descriptivos que indiquen el propósito
- **Versiona**: Mantén un control de cambios para plantillas importantes
- **Documenta**: Incluye descripciones detalladas de cada plantilla
- **Categoriza**: Agrupa plantillas por tipo de proyecto o cliente

### Diseño Responsivo

- **Prueba en diferentes formatos**: Verifica cómo se ve en A4, carta, etc.
- **Considera márgenes**: Asegura espacio suficiente para encuadernación
- **Optimiza para impresión**: Usa colores que se vean bien en blanco y negro

### Seguridad y Privacidad

- **Evita datos sensibles**: No incluyas información confidencial en plantillas compartidas
- **Controla acceso**: Establece permisos apropiados para plantillas personalizadas
- **Revisa contenido**: Verifica que no haya datos de clientes reales en plantillas globales

## Solución de Problemas

### Errores de Renderizado

Si una plantilla no se muestra correctamente:

1. Verifica que todas las variables estén correctamente escritas
2. Asegúrate de que las ayudas tengan los parámetros correctos
3. Comprueba que las estructuras condicionales estén bien formadas

### Problemas de Formato

Para resolver problemas de formato:

1. Revisa el uso de espacios y sangrías
2. Verifica que las etiquetas HTML estén correctamente cerradas
3. Asegúrate de que el CSS sea compatible con la generación PDF
